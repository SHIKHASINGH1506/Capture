## **FEATURES**
Features üöÄ 
  - Authentication
      - Signup
      - Login & logout
  - Feed with posts of user and other people user has followed 
  - Explore with all the posts
  - Add, edit and delete post
  - Like/unlike, bookmark and comment post
  - Edit user profile
  - Follow and unfollow people in suggestions
  - Search user
